# Battlebots

A fun, interactive game where you control a bot and fight against an enemy bot.

## Features

*   Control your bot with the arrow keys.
*   Shoot bullets at the enemy bot using the spacebar.
*   Engage in melee attacks with the 'Z' key.
*   Real-time health and score tracking.

## Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd battlebots
    ```
3.  Install the dependencies:
    ```bash
    npm install
    ```

## How to Run

1.  Start the server:
    ```bash
    node server.js
    ```
2.  Open your web browser and go to `http://localhost:3000` to play the game.
